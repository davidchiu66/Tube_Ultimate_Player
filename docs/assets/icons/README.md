# Tube_Ultimate_Player Icon Set

This directory contains the application icon assets.

- `app-icon.svg`: editable vector source.
- `app-icon.ico`: Windows executable, installer, title bar, and desktop shortcut icon.
- `app-icon.png`: 512 px transparent PNG for documentation and about pages.
- `app-icon-titlebar.png`: 32 px transparent PNG for compact title bar or dialog use.
- `app-icon-about.png`: 256 px transparent PNG for an about page.
- `app-icon-*.png`: exported PNG sizes for UI and packaging variants.

The visual direction is a futuristic neon play-core mark with a hexagonal tech frame and orbital motion lines.
