# Tube_Ultimate_Player 0.2.6

`0.2.6` 是一次围绕播放器操作效率、自动连播稳定性和界面可读性的增强版本。

## 本版本重点

- 新增完整且可配置的播放器快捷键体系
- 设置页新增独立“快捷键”Tab，支持修改、禁用、冲突校验和恢复默认
- 修复 Bilibili XML 弹幕误作字幕加载时导致界面卡死的问题
- 修复播放列表自动连播下一集加载成功后没有自动播放的问题
- 首页和搜索结果长标题限制为三行，超出部分显示 `...`

## 播放快捷键

- `Space`：播放 / 暂停
- `S`：停止
- `D`：下载当前视频
- `C`：收藏当前视频
- `Ctrl+C`：投屏 / 停止投屏
- `Enter / Return`：全屏 / 退出全屏
- `← / →`：后退 / 前进 10 秒
- `Ctrl+← / Ctrl+→`：后退 / 前进 60 秒
- `↑ / ↓`：音量增加 / 降低 5
- `M`：静音 / 恢复音量
- `Home / End`：跳转到开头 / 结尾
- `PageUp / PageDown`：播放列表上一项 / 下一项

快捷键配置会持久化保存；设置保存后播放器立即重新加载配置。输入焦点位于搜索框或文本编辑控件时，播放器快捷键会自动停用，避免输入过程中误触。

## 自动连播修复

- 修复播放列表开启自动连播后，下一集虽然解析和加载成功，但 libmpv 仍保留上一集 EOF 暂停状态的问题
- 新媒体加载时明确恢复真实播放器状态，不再只更新界面上的播放按钮
- 清晰度切换继续保留用户手动暂停状态；播放结束后切换清晰度仍可从头恢复播放

## Bilibili 字幕修复

- 修复 Bilibili XML 弹幕被错误当作标准字幕交给 libmpv 的问题
- 原实现可能在 UI 主线程执行远程 XML `sub-add` 时长期阻塞，造成整个播放界面无响应
- 字幕解析器只暴露 VTT、SRT、ASS、SSA、TTML、JSON3 和 LRC 等受支持格式
- XML 弹幕在解析层过滤，字幕加载入口同时保留二次保护、日志和中文提示

## 首页与搜索体验

- 首页和搜索页共用的视频卡片标题最多显示三行
- 超出第三行的内容使用 `...` 省略，避免挤压作者和时长信息
- 支持中文连续文本和英文单词换行
- 鼠标悬停标题时仍可查看完整内容
- 卡片尺寸变化时会重新计算省略位置

## 配置与界面

- 默认配置新增全部播放器快捷键映射
- 设置页调整为“常规”和“快捷键”两个 Tab
- 深色主题补充快捷键编辑器、滚动区域和 Tab 控件样式
- 播放器停止或切换媒体时同步重置进度显示，快捷键 Seek 使用最新播放位置和时长
