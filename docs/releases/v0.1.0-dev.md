# Tube_Ultimate_Player v0.1.0-dev

发布日期：2026-07-07

这是 `Tube_Ultimate_Player` 的首个公开预发布版本，覆盖了从 YouTube 浏览、搜索、播放到下载管理的完整主流程，并完成了便携版、安装包与 GitHub Releases 的首轮发布打通。

## 版本亮点

- 基于 `PySide6 + yt-dlp + libmpv` 的 Windows 桌面播放器正式成型
- 支持首页推荐、关键词搜索、URL 直达播放三种主入口
- 支持视频解析、清晰度切换、字幕切换、播放控制与全屏沉浸播放
- 支持下载队列、并发下载、暂停/继续/删除、下载完成回看
- 支持收藏、历史记录、首页/搜索九宫格卡片浏览与分页
- 支持 Cookie、代理、FFmpeg、下载目录、浏览器 Cookie 自动发现等设置
- 完成 `%LocalAppData%\Tube_Ultimate_Player` 运行时目录迁移，避免安装到 `C:\Program Files\...` 时的写权限问题
- 完成便携版、安装包、GitHub Actions 与 GitHub Releases 的自动发布链路

## 当前功能总览

### 1. 视频发现

- 应用启动后默认进入首页，而不是直接进入播放器
- 使用 Cookie 获取 YouTube 首页视频列表
- 首页内容使用分页九宫格/卡片网格展示
- 首页支持刷新、选中播放、双击直接播放
- 首页与搜索结果展示风格保持一致

### 2. 搜索与 URL 播放

- 顶部工具栏支持关键词搜索
- 搜索结果支持分页展示
- 点击“播放URL”会弹出专用输入对话框
- 支持直接输入 YouTube URL 进行解析和播放
- 支持普通视频链接、常见短链与可规范化的 YouTube 链接

### 3. 播放器能力

- 基于 `libmpv` 进行视频播放
- 支持播放、暂停、停止、进度跳转、音量调节、倍速调节
- 支持单击画面播放/暂停切换
- 支持双击画面进入全屏，再次双击退出全屏
- 支持全屏按钮
- 支持全屏和窗口模式下的控制面板自动滑出/隐藏
- 鼠标空闲 3.2 秒后自动隐藏光标与控制面板
- 鼠标移动到控制面板区域时再显示控制面板
- 支持字幕解析与加载
- 支持清晰度/格式自动筛选和回退

### 4. 历史与收藏

- 自动记录播放历史
- 提供独立的历史页面
- 支持收藏视频
- 首页卡片和播放控制区域都可以进行收藏
- 提供独立的收藏管理页面

### 5. 下载管理

- 支持将当前播放视频加入下载队列
- 支持并发下载，默认 1，最大 10
- 支持下载进度、速度、剩余时间展示
- 支持暂停、继续、删除操作
- 支持下载完成 toast 提示
- 支持下载完成后双击或点击“播放”直接使用 `libmpv` 本地回看
- 下载任务可持久化，重启应用后仍可恢复显示

### 6. 设置能力

- 支持代理设置
- 代理读取顺序为：优先系统代理，其次用户配置代理
- 支持 Cookie 文本直接粘贴保存到运行时 `cookie.txt`
- 支持浏览器 Cookie 自动发现，包括系统已安装浏览器与默认浏览器优先标识
- 支持 FFmpeg 目录配置
- 支持下载保存目录配置
- 支持 JS runtime 自动检测与手动配置

### 7. 日志与问题定位

- 启动时自动清空旧日志，只保留本次运行日志
- 自动创建 `logs` 目录并输出详细日志
- 记录 yt-dlp 调用、解析失败、下载状态等关键过程
- 运行时配置、数据库、日志、缓存、下载目录统一放入 `%LocalAppData%\Tube_Ultimate_Player`

## JS runtime 说明

`yt-dlp` 在解析部分 YouTube 视频时，可能需要可用的 JavaScript runtime 来完成签名、播放器脚本或 `nsig` 相关处理。程序会自动检测以下运行时：

- Node.js
- Deno
- Bun
- QuickJS / qjs

如果用户没有安装 Node.js，可按以下方式处理：

1. 访问 Node.js 官网下载安装 LTS 版本：<https://nodejs.org/>
2. 安装完成后重新启动应用
3. 在“设置”中保持 `JS runtime = auto`，程序会自动检测 `node`
4. 如果自动检测失败，可在设置中手动指定可执行程序路径

如果不想安装 Node.js，也可以安装 `Deno`、`Bun` 或 `QuickJS`，程序同样会尝试自动识别。

更多说明见：

- [JS Runtime 使用说明](https://github.com/davidchiu66/Tube_Ultimate_Player/blob/main/docs/js_runtime_guide.md)
