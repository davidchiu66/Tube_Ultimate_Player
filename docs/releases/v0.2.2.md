# Tube_Ultimate_Player 0.2.2

`0.2.2` 主要用于修正正式发布链路中的第三方依赖策略，确保 GitHub Actions 能稳定构建便携版与安装包。

## 本版本修复

- 调整 `3rdpart` 目录策略：
  - 仓库继续保留 `libEGL.dll`、`libGLESv2.dll`、`libmpv.dll.a`、`libvlc.dll`、`libvlccore.dll`
  - 仅排除并在发布时动态获取：
    - `3rdpart/libmpv-2.dll`
    - `3rdpart/yt-dlp.exe`

- 修复 GitHub Actions 发布失败：
  - `mpv-dev-x86_64-v3-20260607-git-71ebd08.7z` 仅用于提取 `libmpv-2.dll`
  - `libEGL.dll` 与 `libGLESv2.dll` 改为直接使用仓库内 `3rdpart` 文件
  - 发布工作流继续校验 `libmpv-2.dll`、`libEGL.dll`、`libGLESv2.dll` 三个关键 DLL 的存在性

## 发布流程

- 后续版本继续统一走正式版本发布流程
- GitHub Release 使用正式版发布，不再标记为 prerelease

## 说明

- 本版本是对 `0.2.1` 发布流程问题的快速修复版
- 目标是恢复 GitHub Releases 便携版与安装包的稳定产出
