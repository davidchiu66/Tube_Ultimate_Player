# Tube_Ultimate_Player 0.2.3

`0.2.3` 是一次面向双站点体验与播放器工作流的功能扩展版本，重点完成了 Bilibili 支持、播放列表体系、关于页升级链路，以及工具栏与设置体验的收口。

## 本版本重点

- 新增 Bilibili 首页、搜索、播放、下载与列表类内容支持
- 新增播放器内播放列表面板与独立列表详情页
- 默认首页可在 `Bilibili / YouTube` 之间切换，保存后立即生效
- 设置页增强：浏览器 Cookie 自动获取、Cookie 文本、FFmpeg、JS Runtime 安装入口
- 新增关于页：版本检测、Release Note 查看、在线升级下载
- 顶部工具栏完成新一轮视觉重构与高 DPI 适配

## 新增功能

### Bilibili 支持

- 支持 Bilibili 首页推荐加载
- 支持 Bilibili 关键词搜索
- 支持 Bilibili 单视频 URL 解析与播放
- 支持 Bilibili 视频下载
- 支持以下列表类内容：
  - 多 P 视频
  - 番剧 `ep` / `ss`
  - 稍后再看
  - 收藏夹 / 媒体列表
  - 空间合集 / season 列表

### 播放列表能力

- 新增播放列表详情页
- 新增播放器右侧滑入式播放列表面板
- 支持命名保存、重新加载、删除命名播放列表
- 支持自动连播
- 支持 `Ctrl` 多选与批量下载
- 支持当前播放项高亮与双击切换播放

### 设置与系统集成

- 默认首页切换为单选配置，保存后立即刷新首页行为
- Cookie 改为文本框编辑并写回 `cookie.txt`
- 支持自动检测系统已安装浏览器并提取 Cookie
- 新增 FFmpeg 目录配置
- 新增 JS Runtime 检测与安装入口

### 关于与升级

- 新增内嵌关于页，替代旧弹窗
- 展示应用版本、安装模式、GitHub 链接
- 支持检测远程版本
- 支持查看远程 Release Note
- 支持下载便携版或安装包升级文件，并显示下载进度

## 体验优化

- 顶部工具栏按播放器软件风格重构，统一图标、按钮尺寸与深色主题
- 搜索、首页加载、解析播放 URL 时增加等待动画与提示语
- 首页与搜索结果统一为分页九宫格样式
- 播放控制面板与鼠标隐藏逻辑继续优化，提升全屏沉浸感
- 下载列表支持进度、速度、剩余时间与本地播放入口

## 修复与兼容

- 修复 Bilibili 首页与搜索请求 `-400` 的批量加载问题
- 修复默认首页配置保存后未立即生效的问题
- 修复 Bilibili 列表 URL 在部分场景下无法识别的问题
- 修复下载记录持久化、恢复与播放联动中的若干问题
- 修复关于页版本检测链路与发布流程联动问题

## 发布说明

- 本版本继续使用正式版本发布流程
- GitHub Release 会同步上传便携版与安装包
- `yt-dlp.exe` 在工作流中动态下载
- `libmpv-2.dll` 在工作流中从 SourceForge 拉取，其余依赖继续来自仓库内 `3rdpart/`
