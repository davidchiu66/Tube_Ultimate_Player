# Tube_Ultimate_Player 0.2.0-dev

首个 `0.2.0-dev` 开发预览版，聚焦于把播放器从“可用”继续推进到“更像完整桌面产品”。

## 本版本亮点

- 新增 YouTube 首页推荐、搜索结果分页与九宫格展示
- 新增收藏、历史、下载列表与本地回放能力
- 新增下载并发、保存路径、FFmpeg 目录等设置项
- 新增“关于 / 检测版本 / 在线升级 / Node.js 安装引导”入口
- 顶部工具栏升级为现代化深色播放器风格
- 所有运行期配置、日志、数据库、下载目录统一迁移到 `%LocalAppData%\\Tube_Ultimate_Player`

## 新增功能

### 播放与浏览

- 支持通过 URL 解析并播放 YouTube 视频
- 支持加载 YouTube 首页推荐视频
- 支持关键词搜索与结果分页
- 首页与搜索结果均支持卡片式网格浏览
- 支持清晰度切换、字幕切换、音量与倍速控制
- 支持全屏播放、双击全屏、单击播放/暂停

### 收藏与历史

- 新增收藏列表，可从首页与播放器中收藏视频
- 新增历史记录列表，保留播放痕迹并支持再次播放

### 下载能力

- 新增下载队列、暂停、继续、删除与本地播放
- 支持设置视频保存路径
- 支持设置同时下载视频数
- 支持已完成下载视频的本地 libmpv 回放

### 设置与环境

- 支持文本框直接粘贴 Cookie，并保存到运行目录 Cookie 文件
- 支持自动探测系统已安装浏览器及默认浏览器 Cookie 来源
- 代理读取顺序调整为：优先系统代理，其次手动配置代理
- 支持指定 FFmpeg 目录
- 新增 JS Runtime 检测状态与 Node.js 安装引导入口

### 版本与升级

- 新增“关于”页
- 支持检测 GitHub Release 最新版本
- 支持查看 Release Note
- 支持下载便携版或安装包进行在线升级

### UI 与体验

- 顶部工具栏按现代播放器风格重构
- 搜索框、按钮、提示与深色主题统一优化
- 改进等待状态提示与日志输出，便于问题定位

## 工程与发布

- 应用名统一为 `Tube_Ultimate_Player`
- 日志按单次运行清空重写
- 运行期数据统一写入 `%LocalAppData%\\Tube_Ultimate_Player`
- 发布工作流支持构建便携版与安装包，并自动发布到 GitHub Releases
- 发布阶段不再依赖仓库内提交 `3rdpart` 二进制，工作流中动态下载 `yt-dlp` 与 `libmpv`

## 已知说明

- 当前版本仍为开发预览版（`dev`），适合验证功能链路与发布流程
- 在线升级第一版仅完成“下载升级包并提示后续操作”，暂未引入独立热替换 updater
- 若本机未安装 Node.js，涉及 JS Runtime 的部分能力会在设置页中给出引导
